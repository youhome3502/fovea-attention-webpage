Abstract

This research investigates bio-inspired fovea attention mechanisms in deep neural networks for visual classification. We implement and compare multiple approaches: fixed fovea positioning, learnable attention, Vision Transformers with uncertainty estimation, and reinforcement learning-based active gaze control.

Our experiments on CIFAR-10 demonstrate that RL-based active attention achieves 84.79% accuracy, a +2.59% improvement over the ResNet18 baseline (82.20%). Notably, our analysis reveals that with extended training (20 epochs), the baseline itself achieves strong performance, making the improvement margin more modest but more meaningful.

Key Contribution: End-to-end RL training of gaze policy enables dynamic attention positioning that outperforms fixed and learnable alternatives.

Introduction

The human visual system processes information with remarkable efficiency through foveal vision — a small central region provides high-acuity perception while peripheral vision maintains broader awareness at lower resolution. This biological architecture raises a fundamental question: can artificial neural networks benefit from similar selective attention mechanisms?

Traditional convolutional neural networks process all image regions equally, treating every pixel with equivalent computational resources. This approach is both computationally inefficient and biologically implausible. Humans naturally focus computational resources on informative regions, a principle known as visual attention.

This research systematically explores fovea-inspired attention mechanisms across multiple paradigms:

  • Fixed fovea: Always attend to image center at high resolution
  • Learnable attention: Network learns optimal gaze positions
  • Uncertainty estimation: Quantify prediction confidence
  • RL-based active attention: Optimize gaze through reinforcement learning
Research Question: Can bio-inspired attention mechanisms improve deep learning performance on visual classification tasks?

Related Work

Attention Mechanisms in Vision

Attention mechanisms have become fundamental in modern computer vision. From Squeeze-and-Excitation networks to Vision Transformers, the field has explored various ways to weight feature importance. Our work specifically focuses on spatial attention — determining which image regions deserve computational focus.

Biologically Inspired Vision

Fovea-inspired architectures draw from biological vision research. The mammalian visual system allocates disproportionate cortical resources to foveal representations. Computational models attempting to replicate this include:

  • GlanceNet and similar glimpse-based models
  • Recurrent Attention Models (RAM)
  • Spatial transformer networks

Reinforcement Learning for Attention

RL provides a natural framework for gaze control where the reward signal (classification accuracy) directly optimizes attention positioning. Prior work demonstrates that policy gradient methods can learn effective visual search strategies.

Method

M2: Fixed FoveaResNet

Two-stream architecture processing center and periphery separately:

  • Center path: 8×8 patch at full 32×32 resolution (6.25% of pixels)
  • Periphery path: Downsample 2×, process, upsample back
  • Fusion: Element-wise addition after dimension matching

Result: 81.93% (20 epochs)

M9: Learnable Fovea with GlimpseLocator

Network learns optimal gaze positions through soft-argmax differentiable attention:

  • Spatial attention produces probability distribution over locations
  • Soft-argmax provides differentiable center estimation
  • Gaze position guides fovea placement

Result: 75.74% (20 epochs)

M11: UncertaintyViT

Vision Transformer with Bayesian uncertainty via MC Dropout:

  • Patch embeddings + transformer blocks
  • 5 forward passes with dropout enabled
  • Mean = prediction, Std = confidence

Result: 72.40% (20 epochs)

M14: RL Gaze Control (Ours)

Proximal Policy Optimization for active attention:

  • Policy Network: Encodes features, outputs gaze coordinates
  • Algorithm: PPO with clipped surrogate objective
  • Reward: Classification accuracy
  • Exploration: Entropy bonus for diverse strategies

Result: 84.79% (20 epochs)

Results

Key Findings

1. RL-based active attention achieves best performance.

M14 with PPO (84.79%) outperforms the baseline (82.20%) by +2.59%. The network learns to dynamically position gaze based on image content.

2. Fixed fovea is competitive with baseline.

M2 FoveaResNet (81.93%) achieves comparable performance to the plain ResNet18 baseline (82.20%), demonstrating that fovea attention maintains accuracy while potentially reducing computation.

3. Learnable attention underperforms fixed alternatives.

Surprisingly, M9 Learnable (75.74%) performs worse than fixed fovea (81.93%). Analysis shows the network converges to near-center positions anyway, suggesting that for CIFAR-10 with centered objects, fixed attention is sufficient.

4. Extended training significantly impacts results.

Training duration matters critically. The baseline improved from 60% (5 epochs) to 82.20% (20 epochs). Fair comparison requires consistent training length.

Conclusion

This research systematically explores bio-inspired fovea attention mechanisms for visual classification. Our key findings:

  1. RL-based active attention is effective: End-to-end reinforcement learning enables the network to learn optimal gaze strategies, achieving 84.79% accuracy (+2.59% over baseline).
  2. Fixed fovea provides competitive performance: Simple fixed-center attention achieves 81.93%, comparable to the plain baseline while potentially reducing computation.
  3. Optimal fovea size exists: 8×8 center (6.25% of 32×32 image) provides the best balance between detail and context.
  4. Training duration matters: Fair comparison requires consistent training length across all models.

Final Result

Our RL-based fovea attention achieves 84.79% accuracy on CIFAR-10, demonstrating that bio-inspired active attention mechanisms can improve visual perception in deep neural networks.
