Project Overview

The human visual system processes information with remarkable efficiency. The fovea - a small region in the center of the retina - provides high-acuity vision for detailed tasks, while peripheral vision provides broader context at lower resolution.

Core Hypothesis: Fovea-inspired attention mechanisms can significantly improve deep learning models.

Module Details

M1: Plain ResNet18 Baseline60.00%

Research Motivation: Every meaningful experiment requires a control group. Plain ResNet18 serves as the essential reference point without any attention mechanisms.

Technical Principle: ResNet18 consists of 4 residual blocks with skip connections, progressively increasing channels (64, 128, 256, 512) while reducing spatial dimensions.

Results: 5 epochs: 60%. Extended 20 epochs: 82.20%. This demonstrates that training duration is crucial for fair comparison.

Key Insight: Plain ResNet18 treats all image regions equally with no selective attention, establishing the baseline our fovea approaches must exceed.

M2: FoveaResNet (Fixed Center)68.99%

Research Motivation: Humans evolved the fovea because not all visual information is equally important. This tests if mimicking biological foveal architecture improves neural networks.

Technical Principle: Two parallel streams: (1) Center path extracts high-resolution features from 8x8 central patch (6.25% of image). (2) Periphery path downsamples 2x, processes, upsamples. Features combined via element-wise addition.

Bug Fix: Initial implementation used torch.zeros_like(x) which destroyed peripheral features. Spatial masking preserves periphery while adding high-resolution center.

Results: +8.99% improvement. Optimal center size of 8x8 (6.25%) is consistent with human vision.

M9: Learnable Fovea with GlimpseLocator63.63%

Research Motivation: If fixed center improves performance, perhaps optimal positioning could improve further. Can the network learn to predict where to look?

Technical Principle: GlimpseLocator processes feature maps through spatial attention. Soft-argmax computes differentiable "center of attention" for backpropagation.

Results: Surprisingly, learnable fovea (63.63%) < fixed center (68.99%). GlimpseLocator converges to near-center locations, suggesting for CIFAR-10, optimal gaze is typically centered.

Key Finding: For CIFAR-10 with centered objects, fixed center is already near-optimal.
Module 3: UncertaintyViT with MC Dropout72.40%

Research Motivation: Can models quantify their uncertainty? In critical applications, knowing when the model is uncertain is as important as correct predictions.

Technical Principle: Monte Carlo Dropout runs multiple forward passes with dropout enabled. Mean = prediction, Std = uncertainty. Approximates Bayesian inference.

Results: 72.40% accuracy, +12.4% improvement. Uncertainty values increase during training (0.16 to 0.38), suggesting better calibration.

Key Finding: ViT architecture outperforms ResNet, and uncertainty provides confidence without accuracy penalty.
Module 4: MRAM (Multi-Resolution Adaptive Mapping)66.06%

Research Motivation: What if we process multiple resolution levels simultaneously? 3-level pyramid (1x, 2x, 4x) with learnable fusion weights.

Technical Principle: Three parallel streams process at different resolutions. Learnable softmax-normalized weights determine fusion: w1=0.61, w2=0.23, w3=0.17.

Results: 66.06% accuracy. Learned weights show 61% preference for highest resolution, confirming high-res features are most informative.

M14: RL-Based Gaze Control (BEST)84.80%

Research Motivation: The ultimate goal is active, intelligent gaze selection. Can the system learn to look at the most informative image regions? Applies reinforcement learning (PPO) to optimize fovea positioning.

Technical Principle: Proximal Policy Optimization (PPO) with clipped surrogate objective enables stable policy learning. Policy outputs continuous gaze coordinates. Reward = classification accuracy. Exploration encouraged through entropy bonus.

Results: 84.80% accuracy, +24.8% improvement. Best result across all modules.

Key Finding: RL-based active gaze vastly outperforms all passive methods. The network learns dynamic attention positioning.
